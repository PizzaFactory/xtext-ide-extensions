## Known Issues

## Development Hints

The Textmate "grammar" was developed using the syntax highlighting grammer edior [*Iro*](https://eeyo.io/iro/). For working on the TextMate "grammar" please consider using the *Iro* specifcaton file ../Xtext.iro.
